# React-Redux-Starter
React, redux, enzyme project following Cory House's course https://app.pluralsight.com/library/courses/react-redux-react-router-es6

#Requirements
[Node.js](https://nodejs.org/en/download/)

#Instructions
1. Navigate to the project location folder
2. Open cmd prompt in that location
3. Type "npm install" and hit enter
4. After installation, type "npm run build"
5. The tests, linting, and bundling will start and after a few seconds the site will show up on your default browser
